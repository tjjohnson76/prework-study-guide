HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title for a webpage. (title in tab bar)
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.
  • Elements that are nested inside of other elements are known as child elements.

CSS

  • Style sheets are typically set aside in a separate file(.css) to keep your code organized.
  • A margin property indicates how much space we want around the outside of an element.
  • The padding property adds space around the content inside an element.(increasing padding increases the size of the element)
  • Colors can be added to CSS by using semantic references(red,blue) or by using its hexadecimal code.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
    • Git flow for starting branch
    1. navigate to the working directory
    2. enter "code ." to open vs code of your directory
    3. git checkout main
    4. git checkout -b "branch name"
    • Git flow for commits
    1. git add -A
    2. git commit -m "your message here"
    3. git pull origin main
    4. git push origin "branch name"
    5. navigate to pull requests to github website
    6. git checkout main(start update to main branch)
    7. git pull origin main(pulls updated main branch)

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Right-clicking and selecting inspect on a webpage allows you to use and view the console for the page.
  • constants can be of any variable type, but you need to take care when combining variable of different types.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
  • Functions must be called in order for the code in them to be run.
